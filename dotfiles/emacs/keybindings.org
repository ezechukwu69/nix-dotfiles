## Mode Differences

**Vim** operates in different modes:
- **Normal mode**: Default mode for navigation and commands
- **Insert mode**: For text insertion (`i`, `a`, `o`, etc.)  
- **Visual mode**: For text selection (`v`, `V`, `C-v`)
- **Command mode**: For ex commands (`:`)

**Emacs** is modeless by default:
- Always in "insert mode" equivalent
- Commands are triggered by key combinations
- Regions are selected with `C-SPC` and movement
- Commands are executed with `M-x command-name RET`

## Key Notation

**Vim**:
- `C-x` = Ctrl+x
- `:command` = Ex command mode
- `leader` = Usually `\` or `,` (configurable)

**Emacs**:
- `C-x` = Ctrl+x
- `M-x` = Alt+x (or Esc then x)
- `C-M-x` = Ctrl+Alt+x
- `SPC` = Space bar
- `DEL` = Backspace
- `RET` = Enter
